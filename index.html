HTML as one of my core skills

HTML (HyperText Markup Language) is a markup language that defines the structure and meaning of web content. Unlike programming languages, HTML uses elements (tags) to annotate text, images, and other content for display in browsers.

structural elements

<header>
What it is: Container for introductory content.
What it's used for: Site title, navigation, logos.
When to use: At the top of page or section.
How to use correctly: Place at beginning of page or section.
<nav>
What it is: Section for navigation links.
What it's used for: Main site menus.
When to use: For primary page navigation.
How to use correctly: Wrap major link blocks.
<main>
What it is: Dominant content of the page.
What it's used for: Unique page content.
When to use: Once per page, wrap core content.
How to use correctly: Don't include sidebars or nav.
<section>
What it is: Thematic grouping of content.
What it's used for: Grouping related content with heading.
When to use: For distinct parts of a page.
How to use correctly: Always include a heading.
<footer>
What it is: Footer for page or section.
What it's used for: Copyright, contact, links.
When to use: At bottom of page or section.
How to use correctly: Contains author info, copyright.
